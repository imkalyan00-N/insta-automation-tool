1. Introduction to Web Development

Idi enti? (What is this?) Websites ni tayaru cheyadam ane process ni Web Development antaru. Diniki main ga 3 languages kavali: HTML, CSS, JavaScript.
Enduku use chestaru? (Why use it?) Prapancham lo evaraina mana information chudadaniki, mana skills chupinchukovadaniki (portfolio) websites use chestaru.
Real-world example: Amazon, Instagram, Google ilanti websites anni web development tho ne chestaru.
Syntax:

Website = HTML (Structure) + CSS (Design) + JS (Action)

Code Example:

Code em ledu, basic theory idi.

Prathi line explanation:

Building tho compare cheste: HTML ante itukalu (bricks), CSS ante paint, JS ante current wires & switches.

Common mistakes: Direct ga design (CSS) ki vellipovadam. First HTML correct ga nerchukovali.
Tips: Prathi roju koddiga code type chestu practice cheyi, chadavadam kante type cheyadam best.
Exercise: Web development lo unde 3 main languages perlu rayi.
Exercise Solution: HTML, CSS, JavaScript.

2. What is HTML?

Idi enti? (What is this?) HTML ante HyperText Markup Language. Idi website ki skeleton (asthipanjaram) lanti di. Text, images ekkada undalo idi cheptundi.
Enduku use chestaru? (Why use it?) Browser (Chrome, Edge) ki mana content ni ela chupinchalo cheppadaniki.
Real-world example: Oka book lo heading, paragraph, photos ela untayo, website lo kuda alage pettadaniki HTML vadataru.
Syntax: 
<tagname> Content goes here </tagname>
Code Example: 
<h1> Naa Peru Kalyan </h1>
Prathi line explanation:
  • <h1> - Idi start tag. Heading start ayindi ani cheptundi.
  • Naa Peru Kalyan - Idi mana website lo kanipinche text.
  • </h1> - Idi end tag (slash untundi). Heading aipoindi ani cheptundi.
Common mistakes: End tag (slash /) marchipovadam. Example: <h1> Hello <h1> (idi wrong).
Tips: Tags eppudu chinna aksharalalo (lowercase) rayadam manchi alavatu.
Exercise: Oka basic tag ni start chesi, end chesi rayi.
Exercise Solution: <p> Hello World </p>

3. HTML Structure

Idi enti? (What is this?) Prathi HTML file ki oka fixed format untundi. Daanni boilerplate antaru. Manam ekkada start cheyalo idi cheptundi.
Enduku use chestaru? (Why use it?) Browser ki idi oka HTML page ani, deeni lopala main content undi ani ardam avvadaniki.
Real-world example: Oka letter rase mundu 'To', 'From', 'Date' ela rasthamo, idi HTML ki alanti basic format.
Syntax: 
<!DOCTYPE html>
<html>
    <head>...</head>
    <body>...</body>
</html>
Code Example: 
<!DOCTYPE html>
<html>
<head>
    <title>My Website</title>
</head>
<body>
    Hello World!
</body>
</html>
Prathi line explanation:
  • <!DOCTYPE html> - Browser ki idi latest HTML5 ani cheptundi.
  • <html> - Total webpage ikkada start avtundi.
  • <head> - Page gurinchi details (title, settings) ikkada untayi. Bayataki kanipinchavu.
  • <title> - Browser tab meda kanipinche peru.
  • <body> - Manaku screen paina kanipinche content antha dinilone rayali.
Common mistakes: Main content ni <head> tag lopala raseyyadam.
Tips: VS Code editor lo ! kotti Enter nokkite ee structure automatic ga ochestundi.
Exercise: Head lopala title tag rasi nee peru pettu.
Exercise Solution: <head> <title> Kalyan Portfolio </title> </head>

4. Headings

Idi enti? (What is this?) Webpage lo titles mariyu sub-titles chupinchadaniki headings use chestaru. H1 nunchi H6 daka untayi.
Enduku use chestaru? (Why use it?) Content ni sections ga divide cheyadaniki. Pedda title ki H1, chinna title ki H2 vadataru.
Real-world example: Newspaper lo main news pedda ga (H1), lopala unna sub-headings chinna ga (H2, H3) untayi.
Syntax: 
<h1>Main Title</h1>
<h2>Sub Title</h2>
Code Example: 
<h1>Welcome to My Portfolio</h1>
<h2>About Me</h2>
<h3>My Education</h3>
Prathi line explanation:
  • <h1> - Idi sabhapathi lanti main heading. Page ki okate undali.
  • <h2> - Idi H1 kante koddiga chinnadi.
  • <h3> - Idi H2 lopala unde sub-topic ki vadataru.
Common mistakes: Text peddaga kanipinchalani CSS vadakunda <h1> ni prathi chota vadadam wrong.
Tips: Oka page lo eppudu okate <h1> tag undali, Google search ki idi manchidi (SEO).
Exercise: "My Skills" ani oka h2 heading rayi.
Exercise Solution: <h2>My Skills</h2>

5. Paragraphs

Idi enti? (What is this?) Normal text matter rasi, danteelani paragraphs ga divide cheyadaniki <p> tag use chestaru.
Enduku use chestaru? (Why use it?) Nee gurinchi kani, nee projects gurinchi kani detail ga rayadaniki idi vadataru.
Real-world example: Oka story book lo prathi pēra (paragraph) ela gap tho start avtundo alage.
Syntax: 
<p>This is a paragraph.</p>
Code Example: 
<p>Hi, nenu B.Tech 1st year chaduvutunnanu. Naku coding ante istam.</p>
Prathi line explanation:
  • <p> - Paragraph start avtundi ani browser ki cheptundi.
  • Hi, nenu... - Manam chupinchalanukunna text.
  • </p> - Paragraph end aindi ani cheptundi. Next p tag vaste kotha line lo vastundi.
Common mistakes: Lines break cheyadaniki enter kodataru kani code lo enter kottite browser pattinchukodu. <br> tag vadali.
Tips: Dummy text kavali ante VS code lo lorem ani type chesi enter kottu, automatic ga paragraph vastundi.
Exercise: Nee goal gurinchi oka paragraph rayi.
Exercise Solution: <p>Na goal oka manchi software engineer avvadam.</p>

6. Links

Idi enti? (What is this?) Oka page nunchi inko page ki jump avvadaniki Links vadataru. Dini Anchor tag <a> antaru.
Enduku use chestaru? (Why use it?) Nee portfolio nunchi nee GitHub ki leda LinkedIn ki velladaniki user ki oka click button la ivvadaniki.
Real-world example: Google search lo blue color text paina click cheste inko website ki velthav kada, ade link.
Syntax: 
<a href="url">Click Text</a>
Code Example: 
<a href="https://google.com" target="_blank">Google ki vellu</a>
Prathi line explanation:
  • <a> - Anchor tag start.
  • href="..." - Idi chala important. Ekkadaki vellalo address (URL) ikkada ivvali.
  • target="_blank" - Idi isthe, link kotha tab lo open avtundi. Mana website close avvadu.
  • Google ki vellu - Screen paina user ki kanipinche text idi.
Common mistakes: href spelling tappu rayadam. Leda URL mundu https:// marchipovadam.
Tips: Mana same website lo inko section ki vellali ante href="#section-id" ani vadachu (smooth scroll kosam).
Exercise: Nee Instagram profile ki oka link tayaru cheyi.
Exercise Solution: <a href="https://instagram.com">My Instagram</a>

7. Images

Idi enti? (What is this?) Website lo photos pettadaniki Image tag <img> vadataru.
Enduku use chestaru? (Why use it?) Portfolio lo nee photo chupinchadaniki, leda nee projects vi screenshots chupinchadaniki.
Real-world example: Amazon lo product photo kanipistundi ga, adi image tag thone pettedhi.
Syntax: 
<img src="image_url" alt="description">
Code Example: 
<img src="myphoto.jpg" alt="Kalyan Photo" width="200">
Prathi line explanation:
  • <img> - Image tag (Diniki end tag undadu, deenini self-closing tag antaru).
  • src="..." - Source. Photo ekkada undo aa path leda link ivvali.
  • alt="..." - Alternative text. Oka vela net slow undi photo load avvakapothe, ee text kanipistundi.
  • width="200" - Photo size ni chinna ga cheyadaniki.
Common mistakes: alt attribute marchipovadam. Idi Google ki photo ento ardham avvadaniki chala avasaram.
Tips: Images size thakkuva (compressed) unte website fast ga load avtundi.
Exercise: Logo.png image ni pettu, alt ga "My Logo" ani rayi.
Exercise Solution: <img src="logo.png" alt="My Logo">

8. Lists

Idi enti? (What is this?) Items ni line ga chupinchadaniki Lists vadataru. Rendu types: Ordered List (Numbers 1,2,3) mariyu Unordered List (Bullet points).
Enduku use chestaru? (Why use it?) Nee skills ni (ex: HTML, CSS, JS) okadanikinda okati bullets lo chupinchadaniki.
Real-world example: Kirana shop list (1. Rice, 2. Dal) laaga chupinchali ante List tag vadali.
Syntax: 
<ul>
    <li>Item</li>
</ul>
Code Example: 
<h3>My Skills:</h3>
<ul>
    <li>HTML</li>
    <li>CSS</li>
</ul>
Prathi line explanation:
  • <ul> - Unordered List start. (Chukkalu vastayi). Numbers kavali ante <ol> rayali.
  • <li> - List Item. Prathi skill ni oka li lopala rayali.
  • </li> - Item aipoindi.
  • </ul> - List close aindi.
Common mistakes: Direct ga <ul> tag lopala text raseyyadam, <li> vadanikunda.
Tips: Navigation menu (Home, About, Contact) tayaru cheyadaniki kuda ee ul, li tags ne vadataru.
Exercise: 1, 2, 3 numbers tho Top 2 subjects rayi.
Exercise Solution: <ol> <li>Maths</li> <li>Physics</li> </ol>

9 & 10. Tables & Forms (Basics)

11 to 17. Website Sections (Header, Nav, Hero, About, Skills, Projects, Contact)

Idi enti? (What is this?) Oka manchi portfolio ki konni main bhagalu (sections) untayi.

1. Header/Nav: Top lo unde menu (Home, About).
2. Hero Section: Open cheyyagane first kanipinche pedda introduction.
3. About Section: Nee gurinchi details.

6. Contact Section: Ninnu ela reach avvalo cheppe form leda details.
Enduku use chestaru? (Why use it?) Antha okate deggara kakunda, neat ga categorize cheste chudevallaki ezy ga ardham avtundi.
Code Example: 
<header> <nav>Menu Items</nav> </header>

<section id="hero"> 
    <h1>Hi, I am Kalyan</h1> 
</section>

<section id="about"> ... </section>

<section id="contact"> ... </section>
Prathi line explanation:
  • <section> - Idi div lanti tag ae, kani web page lo oka main part ani ardham.
  • id="hero" - Ee section peru "hero" ani istunnam. CSS apply cheyadaniki easy ga untundi.
Tips: Prathi section ki oka unique ID isthe, Navbar lo click chesinappudu akkadaiki smooth ga scroll avvochu.

18. CSS Basics

Idi enti? (What is this?) CSS ante Cascading Style Sheets. HTML website ki makeup veyadam lanti di (colors, sizes, layout).
Enduku use chestaru? (Why use it?) Black and white html page ni andam ga, professional ga marchadaniki CSS vadataru.
Real-world example: Illu kattaka dానికి paint veyyadam, tiles add cheyadam alantidi CSS.
Syntax: 
selector {
    property: value;
}
Code Example: 
h1 {
    color: blue;
    text-align: center;
}
Prathi line explanation:
  • h1 - Selector. (Deniki design ivvali? h1 tag ki).
  • { ... } - Ee bracket lopala design rules rastaam.
  • color: blue; - Text color blue avvali ani cheptunnam.
  • text-align: center; - Text center lo undali ani cheptunnam.
Common mistakes: Semi-colon ; line chivara marchipovadam. Appudu next code pani cheyadu.
Tips: CSS rayadaniki 3 ways unnayi: Inline (tag lopala), Internal (head lopala style tag), External (.css file). External best.

19 & 20. Colors and Fonts (CSS)

Idi enti? (What is this?) Background colors, text colors marchadaniki mariyu text style (Font) marchadaniki.
Code Example: 
body {
    background-color: #f4f4f4;
    color: #333333;
    font-family: Arial, sans-serif;
    font-size: 16px;
}
Prathi line explanation:
  • background-color: #f4f4f4; - Background light grey avtundi. (# code ni HEX code antaru).
  • color: #333333; - Text color dark grey avtundi.
  • font-family - Text design Arial style loki marutundi.
  • font-size - Aksharala size 16 pixels untundi.
Tips: Google Fonts (fonts.google.com) nunchi free ga manchi andamaina fonts techukoni nee portfolio ki vadochu.

21 & 22. Margin and Padding (Box Model)

Idi enti? (What is this?) Spacing! Elements ki lopalika (Padding) leda bayatiki (Margin) gap ivvadaniki.
Real-world example: Oka photo frame theesko. Photo ki frame ki madhya unna gap 'Padding'. Rendu frames ki madhya unna gap 'Margin'.
Code Example: 
.box {
    padding: 20px;
    margin: 30px;
    border: 2px solid black;
}
Prathi line explanation:
  • padding: 20px; - Box border nunchi lopala unna text ki 20px gap istundi.
  • margin: 30px; - Ee box ki pakkana unna inko box ki madhya 30px duram istundi.
  • border - Box chuttu oka black line istundi.
Exercise: Oka button ki lopala gap peragalante em vadali? Solution: Padding vadali.

23. Flexbox

Idi enti? (What is this?) Items ni pakkapakkanne (row) leda okadani kinda okati (column) neat ga arrange cheyadaniki vadataru.
Enduku use chestaru? (Why use it?) Website lo sections ni center cheyadaniki leda space ni equal ga panchadaniki. Idi chala powerful.
Code Example: 
.nav-links {
    display: flex;
    justify-content: center;
    gap: 20px;
}
Prathi line explanation:
  • display: flex; - Flexbox on aindi. Lopala items anni oka line loki ochesthayi.
  • justify-content: center; - Items anni madhyaloki (center) vachayi.
  • gap: 20px; - Prathi item ki madhya 20px gap istundi.
Tips: Portfolio lo skills ni oka grid la chupinchadaniki flexbox ne ekuva vadataru.

24. Responsive Design

Idi enti? (What is this?) Website ni Laptop lo, Tablet lo, Phone lo... anni screens lo andam ga kanipinchela cheyadam.
Enduku use chestaru? (Why use it?) Ippudu andaru phones lalo ne websites chustunaru. Phone chinnadi kabatti design adjust avvali.
Code Example: 
@media (max-width: 600px) {
    .container {
        flex-direction: column;
    }
}
Prathi line explanation:
  • @media (max-width: 600px) - Screen size 600 pixels kante thakkuva unte (Phone anuko) ee CSS apply avvali.
  • flex-direction: column; - Pakkapakkana (row) unnavi, okadani kinda okati (column) loki marutayi.

25, 26, 27, 28. JavaScript Basics (Functions, Events, DOM)

Idi enti? (What is this?) JavaScript ante website ki brain. Oka button click cheste em jaragalo idi cheptundi.
Enduku use chestaru? (Why use it?) Dark mode on cheyadaniki, form validate cheyadaniki, alerts ivvadaniki JS kavali.
Real-world example: Car lo html structure ayite, css color ayite... JS ante engine & steering lanti di.
Code Example (Dark Mode Button): 
// 1. Element ni thechukovadam (DOM)
let btn = document.querySelector(".theme-toggle");

// 2. Function rayadam
function changeTheme() {
    document.body.classList.toggle("dark");
}

// 3. Event (Click chesinappudu)
btn.addEventListener("click", changeTheme);
Prathi line explanation:
  • document.querySelector - HTML lo unna button ni vetiki JS loki thesthundi (DOM Manipulation).
  • function() - Oka task cheyadaniki rase code block.
  • classList.toggle("dark") - Body tag ki 'dark' ane class ni add leda remove chestundi.
  • addEventListener("click") - Evaraina button paina click chestaremo ani wait chestundi (Event).
